# 🎓 Gemini Pro 学生版一年免费领取教程

本教程旨在帮助你通过 **Google Student 认证**，免费获取为期一年的 **Gemini Advanced (Pro)** 会员资格。

## 📌 核心准备

在开始之前，请务必确保满足以下条件：

* **纯净美国节点**：这是成功率的决定性因素。建议使用独享 IP 或口碑较好的美区原生 IP。
* **资金准备**：咸鱼购买虚拟 Visa 卡（约 10 元左右），用于绑定验证。

---

## 🛠️ 操作步骤

### 第一步：环境准备

1. 将梯子切换至 **纯净美国节点**。
2. 打开 Chrome 浏览器，进入 **无痕模式 (Incognito Window)**。

### 第二步：注册 Google 新账号

1. 在无痕模式下注册一个新的 Google 账号。
2. **技巧**：新账号通常可以跳过手机号验证；如果提示需要验证，建议使用流量卡接收短信。

### 第三步：访问申请页面

1. 首先尝试访问：[Gemini Students 官网](https://gemini.google/students/)。
2. 如果页面提示“无资格”，请直接点击下方的备用直达链接：
👉 [Google One AI Student 权益页](https://one.google.com/ai-student?g1_landing_page=75&utm_source=antigravity&utm_campaign=argon_limit_reached)

### 第四步：完成 SheerID 学生验证（核心环节）

这一步无需使用繁琐的 Telegram Bot，直接通过网页端工具完成。

1. 打开验证辅助工具：[1Key Batch 验证页](https://batch.1key.me/)。
2. **操作细节**：
* 在页面中获取验证请求后，**直接右键复制 SheerID 链接**（切记不要直接在浏览器打开该链接）。
* 将链接交由工具处理，等待验证完成。


3. **注意**：如果提示失败，请尝试**更换节点**并**重新获取 SheerID 链接**进行多次尝试。
4. 当页面显示 **"Successful"** 时，说明学生身份已通过验证！

### 第五步：绑定支付方式

1. **获取卡片**：在咸鱼搜索“Visa/美区支付”，购买一张金额约 $1 的测试卡（价格通常在 10 元左右）。
2. **绑定卡片**：按照 Google One 支付页面的提示，填写卡号、过期时间和 CVC。
3. 由于已经通过学生认证，此卡仅作身份激活使用，不会扣除大额月费。

完成上述步骤后，你的账号将成功激活 **Gemini Advanced** 权益，有效期通常为 **12 个月**。你可以立即享受超大上下文窗口及最强的模型能力。
